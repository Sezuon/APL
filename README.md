# Atomizer's Platform Library

The main goal of APL is to allow developers to write code that can be compiled for both Windows and Linux. Just by using different build systems!
Compiling for Windows is as easy as just... Compiling. All you need is the WindowsSDK, which is usually provided via Visual Studio or MinGW. Building for Linux is similar. You just need the VulkanSDK available on your Linux machine.

Check out the **Examples** branch to learn more!
